# Copilot Instructions for JavaScript-Learning

When the user writes `check knowledge`, treat it as a request to review the current JavaScript-Learning workspace and identify missing or incomplete learning topics.

Specifically:
- Inspect the `Syntax/` folder structure and any homework or project folders.
- Compare the existing topics against common JavaScript learning areas (basic syntax, arrays, functions, OOP, ES6, DOM, asynchronous JavaScript, APIs, error handling, modules, etc.).
- Tell the user which topics are already covered and which topics or examples are missing.
- Suggest additional topics, examples, or exercises the user could add to complete the learning path.

Example user prompt:
- `check knowledge`
- `check knowledge for my JavaScript learning folder`
- `ดูว่าผมขาดเนื้อหาอะไรในโปรเจคเรียน JavaScript`

Use Thai when replying to the user unless the user specifically asks for English.
